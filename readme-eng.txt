------
to install
------
This is Muan International Airport (Republic of Korea) for FSX & Prepar3D. (MWX / RKJB)

to install, make folder in your "addon scenery" folder in your FSX (or Prepar3D) folder.
then copy this scenery and run FSX (or Prepar3D) add airport in Scenery Library.

* This Scenery is based on Scott Smart's Muan INTL Airport Terrain data.




------
remark
------
If you using FSX, please delete stock-vehicles-x.BGL file in scenery folder.

If you want Runway and Taxiway center line light,
copy bgl file in optional bgl data folder to scenery folder.
